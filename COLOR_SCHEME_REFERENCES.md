# Standard Color Scheme References

## 1. Air Pollution (PM2.5) - EPA AQI(Environmental Protection Agency Air Quality Index system ) Standard Colors

### Official EPA Sources:
- **EPA AirNow Website**: https://www.airnow.gov/aqi/aqi-basics/
  - Shows the official EPA Air Quality Index color scheme
  - Green (Good), Yellow (Moderate), Orange (Unhealthy for Sensitive Groups), Red (Unhealthy), Purple (Very Unhealthy)

- **EPA AirNow Interactive Map**: https://www.airnow.gov/
  - Real-time air quality map using EPA AQI colors
  - Hover over areas to see PM2.5 values and colors

- **EPA AQI Color Reference**: https://www.airnow.gov/sites/default/files/2020-05/aqi-technical-assistance-document-sept2018.pdf
  - Technical documentation with exact color specifications

### Weather Channel:
- **Weather Channel Air Quality Map**: https://weather.com/maps/airquality
  - Uses EPA standard AQI colors for PM2.5

### Exact EPA AQI Colors (from AirNow.gov):
- Green (Good 0-50): #00e400
- Yellow (Moderate 51-100): #ffff00
- Orange (Unhealthy for Sensitive 101-150): #ff7e00
- Red (Unhealthy 151-200): #ff0000
- Purple (Very Unhealthy 201-300): #8f3f97
- Maroon (Hazardous 301+): #7e0023

## 2. Asthma Rates - Health Data Visualization Standards

### CDC Sources:
- **CDC Asthma Data**: https://www.cdc.gov/asthma/most_recent_national_asthma_data.htm
  - Shows standard health data visualization patterns

- **CDC Environmental Public Health Tracking**: https://ephtracking.cdc.gov/
  - Uses standard health data color scales (typically cool-to-warm: blue/green for low, orange/red for high)

- **CDC National Environmental Public Health Tracking Network**: https://www.cdc.gov/nceh/tracking/default.htm
  - Interactive maps with health data using standard color schemes

### Health Data Visualization Standards:
- **CDC Data Visualization Standards**: https://www.cdc.gov/dataviz/index.html
  - Guidelines for health data visualization colors

## Color Verification:

You can verify the colors by:
1. Opening any of these websites
2. Looking at their air quality maps or health data visualizations
3. Using browser developer tools (F12) to inspect the color values used
4. Comparing with the hex codes in the codebase
